3、字符编码：
ASCII 编码：1 个字节，Unicode 编码通常是 2 个字节
UTF-8：可变长编码，一个 Unicode 字符根据不同的数字大小编码成 1-6 个字节，常用的英文字母被编码成 1 个字节，汉字通常是 3 个字节，只有很生僻的字符才会被编码成 4-6 个字节
